# Jenil | Portfolio

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## How to use

1. Clone the repo

```bash
git clone https://github.com/SavaniJenil/portfolio_app.git
```

2. Install dependencies

```bash
npm install
```

3. Start the dev server

```bash
npm run dev
```
